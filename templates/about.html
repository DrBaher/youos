What is YouOS?

YouOS is not a generic AI email writer. It learns from YOUR sent emails — how you phrase things, how long you write, how you sign off — and drafts replies that actually sound like you.

It runs entirely on your Mac. No cloud. No subscriptions. Your data never leaves your machine.

  • Ingests Gmail, Google Docs, WhatsApp exports — plus organic pairs from sent mail without YouOS involvement
  • Handles full email threads — paste the whole thread; same-thread history gets a 2× retrieval boost
  • Per-account corpus isolation — work emails draw from work history; personal from personal
  • Drafts grounded in score-ranked, confidence-annotated, thread-deduplicated exemplars; reply text preserved (600 chars), inbound trimmed (400)
  • Prompt token budget enforced — exemplars auto-trimmed if prompt exceeds 2000 tokens
  • Confidence thresholds are relative to retrieval score distribution (mean±σ), not hardcoded
  • Detects multi-intent (e.g. urgent + approval) and boosts matching exemplars 1.2×; per-intent reply length calibrated from your corpus
  • Per-sender-type persona variants — different voice, length, greeting, and closing for internal, external, and personal contacts
  • Greets people by first name, closes in your style — greeting and closing injected from persona per contact type
  • Richer persona: bullet point rate, directness score, sentence length, confidence intervals (p25/p75) — all used in prompts
  • Persona re-analysis is incremental (recent 90 days × 3 weight), EWMA for avg reply length, full weekly refresh
  • Subject line + topic-aware retrieval signals — known topics from sender profiles boost matching pairs
  • FTS queries expanded with email vocabulary synonyms (reschedule → postpone, meeting, call…)
  • Confidence-gated warnings — low-precedent drafts show a clear banner; explain any draft inline
  • Local model empty or signature-only output → automatic Claude fallback
  • Subject line generated via smart content analysis — skips greeting/filler lines, finds the topic sentence
  • Training data quality filters + deduplication by inbound similarity (≥0.95 → keep higher-rated pair)
  • Curriculum learning for LoRA fine-tuning — quality-ordered warmup before temporal training
  • DPO preference pairs supported — chosen vs rejected contrastive training data
  • Hybrid token + character similarity for smarter auto-feedback capture
  • Auto-calibrating similarity threshold — drops to 0.65 for small corpora
  • Auto-scales LoRA training hyperparameters as your corpus grows
  • Smart nightly pipeline — skips steps when data is insufficient (no wasted compute)
  • Golden eval runs nightly after fine-tuning — composite score tracked in pipeline log
  • Autoresearch benchmarks rotate weekly — prevents overfitting to fixed test cases
  • Persona re-analyzed nightly — all style signals auto-merge into persona config
  • Style drift detection — Stats dashboard flags when your writing patterns shift significantly
  • Sender profiles track reply-time patterns and topics — both used in retrieval boosting
  • Sender notes immediately rebuild that sender's profile: youos note
  • Submit feedback directly from terminal: youos feedback
  • Review queue: A/B comparison between adapter and base model; configurable batch size (5/10/20)
  • Gmail bookmarklet: floating side panel — generate draft, insert into compose, submit star-rated feedback without leaving Gmail
  • History tab: intent and confidence badges on every past draft
  • Optional cross-encoder reranking for higher retrieval precision
  • Corpus health at a glance: youos corpus
  • Golden benchmark suite (10 curated cases): youos eval --golden
  • Setup wizard asks for internal domains — accurate internal vs external classification from day one
  • Rate-limited draft generation — your data is never over-shared
  • Mobile-friendly UI — use the Review Queue anywhere
  • Pre-flight health check: youos doctor
  • Verbose pipeline: youos improve --verbose

Autoresearch

YouOS uses an autoresearch loop inspired by Andrej Karpathy's autoresearch project. Every night, it:

  1. Ingests your new sent emails, Docs, and WhatsApp messages from the last 48 hours
  2. Captures feedback pairs (comparing what YouOS would have drafted vs what you actually sent)
  3. Fine-tunes the local Qwen model if enough new pairs exist
  4. Runs 80 optimisation experiments — testing retrieval weights, sender-type boosts, prompt variants, and composite score weights
  5. Keeps changes that improve your benchmark score, reverts changes that don't
  6. Composite score weights are configurable in configs/autoresearch.yaml — tune pass_rate vs keyword vs confidence emphasis
  7. Writes a structured JSONL run log — benchmark trends visible in the Stats dashboard
  8. High-quality feedback pairs (high rating, low edit distance) surface higher in future retrievals automatically
  9. Auto-feedback lookback window adapts to actual elapsed time — no missed pairs after a gap

The result: YouOS gets a little better every night, automatically.

FAQ

Does YouOS read my emails automatically?
Only sent mail, only when you run setup or nightly ingestion. It never reads in real-time.
What if I want to delete my data?
Run youos teardown. Removes DB, model, and raw cache.
Can I use this without Apple Silicon?
Currently requires Apple Silicon (M1/M2/M3/M4) for the local model. The retrieval and drafting with Claude fallback work on any Mac.
Is this an OpenClaw skill?
Yes. Install via clawhub install youos or from github.com/DrBaher/youos.
Can I use WhatsApp messages to train my model?
Yes. Export a WhatsApp chat as a .txt file and run youos ingest --whatsapp ~/path/to/chat.txt. YouOS parses your messages and pairs them with inbound messages to build reply pairs.
Can I paste a whole email thread?
Yes — the Draft tab supports full threads. YouOS extracts the most recent message and uses the thread history as context automatically.
How do I check if everything is set up correctly?
Run youos doctor. It checks Python version, gog CLI, mlx_lm, your config, disk space, and more — with green/red output.
What does "low confidence" mean in a draft?
It means YouOS couldn't find strong precedents in your corpus — no similar past replies to ground the draft in. The draft was generated using the base model and your persona config, but may not reflect your style as accurately. Build up your Review Queue to improve coverage.
How does YouOS decide which past replies to use?
It combines BM25 full-text search with semantic embedding similarity, then applies sender-type boosts (e.g. external clients rank higher for external client emails) and a quality score derived from your past feedback ratings and edit distance. Replies you approved with minimal edits surface higher over time.
How can I test if YouOS is improving?
Run youos eval --golden against the built-in golden benchmark suite — 8 curated test cases covering work, personal, multilang, and brevity scenarios. Scores are saved to var/golden_results.json so you can track improvement over time.
How do I see corpus health at a glance?
Run youos corpus. It shows reply pair count, document count, feedback pairs, embedding coverage %, quality score distribution (min/median/max), and your top senders by pair count. Use --json for machine-readable output.
How does YouOS tune its own autoresearch weights?
The composite benchmark score weights (pass rate, keyword hit rate, confidence) are stored in configs/autoresearch.yaml and are themselves mutable surfaces in autoresearch. You can also edit them directly to emphasise what matters most for your use case.
Does YouOS write differently to different people?
Yes. YouOS has per-sender-type persona variants — it uses a shorter, more casual tone for internal colleagues, a more professional tone for external clients, and a warm tone for personal contacts. These defaults are set in configs/persona.yaml under modes: and are updated nightly from your corpus patterns.
How does YouOS understand what kind of email it is?
It classifies the inbound email intent — meeting request, approval needed, status update, complaint, introduction, etc. — and uses that to boost the most relevant exemplars from your past replies. The detected intent is shown in the "How was this generated?" panel.
How can I see why a draft was generated the way it was?
Click "How was this generated?" below any draft. It shows the confidence level, detected intent, and the top 3 exemplars that influenced the draft — including their retrieval scores and subject lines. Great for debugging and building trust in the system.
What is style drift and how does it affect me?
Style drift means your recent writing patterns have shifted noticeably from your historical baseline — e.g. your replies are getting significantly shorter, or your directness score has dropped. YouOS tracks this in the Stats dashboard and flags it so you can decide whether to let the persona config adapt or keep the original settings.
Does YouOS know what kind of email I'm replying to?
Yes. YouOS detects up to 3 simultaneous intents per email (e.g. meeting request + urgent) and uses them to boost the most relevant exemplars. It also calibrates reply length per intent — your typical "thank you" reply is shorter than your "status update" reply, and the model learns this from your corpus automatically.
Does YouOS learn from emails I send without using it?
Yes — organic pair capture. Each night, YouOS also collects emails you sent without asking for a draft. These become training pairs automatically, expanding your corpus even on days you don't use the Review Queue.
What happens on quiet days when there's nothing new to learn?
The nightly pipeline has smart skip gates. If fewer than 3 new feedback pairs exist, fine-tuning is skipped. If fewer than 5, autoresearch is skipped. If all documents already have embeddings, the indexer is skipped. This prevents wasted compute and avoids degrading the model on insufficient data.
How do I know if my LoRA adapter is actually helping?
Use the Compare button in the Review Queue. It now runs your Qwen+LoRA adapter against the base Qwen model (no adapter, no exemplars) and shows both drafts side by side — with an improvement hint: "Adapter appears to be helping" when the drafts differ significantly.
Does YouOS keep separate drafting history for work vs personal emails?
Yes. Per-account corpus isolation means that when you draft a reply to a work contact, YouOS retrieves exclusively from your work Gmail history. When replying to a personal contact (gmail.com, icloud.com etc.), it draws from your personal account history. This ensures you don't get work-style drafts for personal emails.
Can I submit feedback without using the web UI?
Yes. Run youos feedback --inbound "email text" --reply "your reply" --rating 4. It inserts directly into the feedback database, same as the web UI. Add --sender for sender context and --note for an optional note. Supports --stdin to pipe text from the terminal.
