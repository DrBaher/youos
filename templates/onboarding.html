✉️ YouOS · Set up

Welcome

A few steps to get YouOS drafting in your voice — all of it runs locally on your Mac. You can change everything later in Settings.

What YouOS does: it reads your past sent emails to learn how you write, then drafts replies that sound like you instead of a generic AI. Nothing leaves your computer — your mail, the model, and the drafts all stay local.

Prefer the terminal? youos setup runs this same flow with each step automated.

Who are you?

So YouOS knows which messages in a thread are yours — and so it can become your OS.

Why this matters: an email thread is a back-and-forth between people. YouOS uses your name and email addresses to pick out your replies (the ones it should learn your style from) versus everyone else's. Add every address you send from — work and personal. Your name also personalizes the app — YouOS becomes YourOS.

Connect Gmail & Docs

Pick how YouOS reaches your mail. It only ever reads to learn your style.

A "backend" is just the tool YouOS uses to read your Gmail & Docs — all three options only read, and only on your machine. Pick one: gog (easiest if you already use OpenClaw), gws (Google's own command-line tool), or native (talks to Google directly, no separate tool to install). Not sure? Leave it on gog.
gog — OpenClaw CLI. Install it, then authenticate each account:
pip install gog-cli
gog auth login
Different setup? See github.com/openclaw/gog.
gws — Google Workspace CLI. Install from github.com/googleworkspace/cli, then:
gws auth login
native — Google API (no external CLI). Install the extra, then authorize each account:
pip install 'youos[google]'
Then set ingestion.google_oauth_client_secrets to your Google OAuth client JSON (Settings or config), and run the authorize step.

Build your corpus

Pull your sent history so YouOS has examples to learn from. More history = a better model, but a longer first run.

Your "corpus" is the collection of your past emails YouOS studies to learn your voice — your greetings, tone, sign-offs, how long you write. The further back you go, the better it captures your style (and the longer this first pull takes). It only reads; nothing is uploaded or sent anywhere.

Runs against your connected account(s) in the background. Prefer the terminal? youos ingest · WhatsApp export: youos ingest --whatsapp chat.txt.

Learn your voice

YouOS fine-tunes a local LoRA adapter on your sent mail so drafts sound like you — we start it automatically here so it's never silently skipped.

"Fine-tuning" teaches a small AI model, running on your Mac, to imitate your writing — using your corpus plus the edits you make to its drafts. (A "LoRA adapter" is just the small file that holds what it learned.) We kick this off automatically in the background when you reach this step — you can keep going through setup; your drafts switch to your fine-tuned voice the moment it finishes. Until then they use the base model + your examples. Nothing is sent to the cloud, and it retrains itself each night as you give feedback.

Needs Apple-Silicon MLX and a handful of pairs; runs in the background (several minutes). Terminal equivalent: youos finetune.

Secure it (optional)

YouOS binds to localhost only. If you expose it (e.g. over Tailscale), protect it with a PIN and/or an API token for the extension.

By default only your own computer can reach YouOS ("localhost"), so you can safely skip this. You only need security if you open it up to your network or a tool like Tailscale. A PIN is a password for the web pages; an API token is a secret key the Gmail extension uses to prove it's allowed to connect.

Stored hashed on disk. Revoke later with youos token-revoke.

A token lets the Gmail extension reach a PIN-protected instance. For a login PIN, see the README's auth section. Skip if you only use YouOS on this machine.

Keep it running

Install YouOS as a background service so it's always available.

Right now YouOS only runs while a terminal stays open. Installing it as a background service (a macOS LaunchAgent) makes it start automatically when you log in, restart itself if it crashes, and survive a reboot — so the Gmail extension and web app are always there. No password or admin rights needed; it still only listens on your own computer.

Terminal equivalent: youos service install · remove anytime with youos service uninstall.
