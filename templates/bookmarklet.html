Draft in Gmail — in your voice

The YouOS extension adds a draft panel right inside Gmail. Install it once; everything stays on your Mac.

✨ Recommended: the browser extension is more reliable than the bookmarklet — no Gmail markup fragility, works with PIN-protected instances, and lets you generate, insert, and rate drafts without leaving your inbox.

Install the extension — Chrome, Edge, or Brave

~1 minute, no app store needed.

  1. Keep your YouOS server running. It already is — this page is served by it.
  2. Open chrome://extensions  (or edge://extensions / brave://extensions).
  3. Turn on Developer mode (toggle, top-right).
  4. Click Load unpacked and select this folder:
    YOUOS_EXTENSION_PATH
  5. Open Gmail and click the teal ✉ button (bottom-right), or the toolbar icon, to open the draft panel.

Different port? Open the extension's Options and set the server URL. PIN-protected instance? Run youos token-create and paste the token into Options (sent as the X-YouOS-Token header, stored only in your browser). Firefox: run ./build-firefox.sh in the extension folder, then load firefox-build/ via about:debugging (experimental).

No-install fallback: the bookmarklet

Prefer not to install an extension? Drag this button to your bookmarks bar, then click it on any open Gmail thread. It can break when Gmail changes its markup — the extension doesn't.
