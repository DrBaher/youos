# Demo Recording Script

## Setup (before recording)
1. Clear browser history/localStorage for YouOS
2. Have a real email ready to paste (a simple business inquiry)
3. Open your local YouOS URL (e.g., http://localhost:8765/feedback)

## Recording flow (aim for 45-60 seconds)
1. [2s] Show the YouOS landing page briefly
2. [3s] Click Review Queue tab — show it loading
3. [10s] Show a pre-loaded email card (inbound + draft side by side)
4. [5s] Edit the draft slightly — show it's editable
5. [3s] Click Submit & Next — show next card loading
6. [5s] Switch to Draft Reply tab
7. [5s] Paste a real email, add sender
8. [8s] Click Generate Draft — show streaming draft appearing
9. [3s] Show the suggested subject line
10. [3s] Show Stats page with corpus count and model status
11. [2s] End on YouOS logo

## Tools to record
- macOS: use QuickTime + Gifski to convert to GIF
- Or: use Kap (https://getkap.co) for direct GIF recording
- Target: 800x600px, 15fps, under 5MB
